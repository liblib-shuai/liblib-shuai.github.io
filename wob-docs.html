📖 概述

AI模型盲测平台是一个专业的Web离线测试工具，用于客观评估不同AI模型的输出质量。通过盲测方式（隐藏模型信息），用户可以专注于输出内容的质量，选择他们认为最好的结果，从而获得更科学、更客观的模型性能评估。

🚀 主要功能

智能测试管理

多文件导入、智能验证、实时状态显示、文件管理

盲测测试系统

随机展示、键盘导航、偏好记录、进度跟踪、断点续测

结果统计分析

传统统计、ELO评分系统、可视化图表、详细记录、结果导出

用户体验优化

响应式设计、媒体预览、视频支持、缩略图预加载、长图支持、本地缓存、结果查看、历史导入

📋 操作指南

选择输出结果

  • 点击选择：点击选择您认为最好的输出结果
  • 图片预览：点击图片可放大查看，支持缩略图快速加载
  • 视频预览：点击视频缩略图可放大播放，视频显示为缩略图并叠加播放图标
  • 长图支持：长图会完整显示，按比例缩放，不会被截断
  • 键盘操作：
    • ← → 快速选择左侧或右侧结果
    • ↑ ↓ 快速导航到上一题或下一题
    • Enter 确认选择并进入下一题
    • Tab 在界面元素间切换焦点

📊 数据格式

输出文件格式

输出文件应为JSON格式，包含模型生成的输出结果：

{
  "test_info": {
    "test_version": "模型版本标识",
    "test_case": "测试用例标识"
  },
  "0": {
    "output_display_info": {
      "output_images": ["输出图片URL1", "输出图片URL2"],
      "output_videos": ["输出视频URL1"]
    }
  }
}

字段说明

  • test_info：测试信息，包含模型版本和用例标识
  • 0, 1等：测试ID，必须与输入文件中的ID对应
  • output_display_info：输出显示信息
    • output_images：输出图片URL数组（支持图片和视频文件）
    • output_videos：输出视频URL数组

视频文件支持

平台支持以下视频格式：

  • MP4：最常用的视频格式
  • AVI：传统视频格式
  • MOV：苹果设备常用格式
  • WMV：Windows媒体格式
  • FLV：Flash视频格式
  • WebM：Web优化格式
  • MKV：开源容器格式
  • M4V：iTunes视频格式

技术特性

  • 异步加载：文件异步加载，不阻塞界面

  • 内存优化：优化内存使用，支持大文件处理
